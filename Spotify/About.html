Spotify clone app


-By Jutt and Sweetie


This project is a Spotify Clone built using HTML, CSS, and JavaScript, designed to mimic the core features of a modern music player. The goal of this project is to understand how a music streaming interface works and to practice implementing real-world features using front-end web technologies.

Key Features

  • Music Player: Play, pause, next, previous, and master play buttons to control tracks.
  • Dynamic UI: Album art, song titles, and “Now Playing” indicators update in real-time.
  • Album & Song Lists: Organized collections of songs that can be played individually.

Future Enhancements

  • Add search & filter functionality for songs/albums.
  • Include a progress bar and time duration tracking.
  • Implement volume control & playlist creation.
  • Connect with a backend (Node.js/Express) for dynamic music storage.
